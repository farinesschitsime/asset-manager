# Asset Manager 

This is an asset management system.

## Technologies

This project uses the laravel framework.

# Installation

## Clone the project

```bash
git clone repo-url
```

### install composer dependencies

```bash
composer install
```

### create the .env file

```bash
cp .env.example .env
```

### generating a key

```bash
php artisan key:generate
```

### configure your environment

in here setup your database credentials such as
- database name
- database host
- database username
- database password

### Run migrations

```bash
php artisan migrate
```

### Seed your database

```bash
php artisan db:Seed
```
